# Roadmap

## Phase 1: Build the Map

- Read 2 VLA surveys and 1 VLN survey
- Build a vocabulary list for VLA, VA, VLN, WAM, action tokenization, affordance, grounding, imitation learning
- Write short notes for each survey

## Phase 2: Core VLA Papers

- RT-1
- RT-2
- Open X-Embodiment / RT-X
- OpenVLA
- pi0
- FAST

Goal: understand how image, language, robot state and action are represented.

## Phase 3: Visuomotor Policy

- Diffusion Policy
- 3D Diffusion Policy
- RDT

Goal: connect VLA with low-level action generation and control.

## Phase 4: VLN and Spatial Reasoning

- VLN survey
- NaVILA
- MapNav
- OmniVLA Navigation

Goal: understand navigation grounding, map memory and spatial reasoning.

## Phase 5: Reproduction

- Run a small LeRobot or Diffusion Policy demo
- Run an OpenVLA or MiniVLA inference demo
- Record environment setup, errors and results under `projects/`
